# 2 递归
## 2.1 何谓递归
- **递归**是解决问题的一种方法，它将问题不断地分成更小的子问题，直到子问题可以用普通的方法解决。
- 通常情况下，递归会使用一个不停调用自己的函数。

### 2.1.1 计算一列数之和
- 循环求和函数
- ```python
  def listsum(numList):
      theSum = 0
      for i in numList:
          theSum += i
      return theSum
  ```
- ```python
  >>> listsum([1, 3, 5, 7, 9])
  25
  ```
- 将问题从求一列数之和重新定义成求数字对之和。
- 数字列表`numList`的综合等于列表中的第一个元素（`numList[0]`）加上其余元素（`numList[1:]`）之和。
